# **App Name**: Life Canvas

## Core Features:

- Blog Post Creation: Admin can create, edit, and delete blog posts with a title, author name, content, and cover image.  Admin to have ability to modify site-wide parameters such as background color and text color.
- Blog Post Storage: Store blog posts, including cover images, and configuration settings in Firestore.
- User Authentication: Admin login page using 'admin' as the username and 'testaccount' as the password, placed at the bottom left corner of the site. Restrict blog creation and editing to authenticated admins.
- Blog Post Display: Display blog posts with title, author, cover image, content, and publication date.
- Search Functionality: Enable users to search for blog posts by keywords.
- Hashtag Support: Automatically create links to search for content using the tag when the author types '#[tagname]'.
- Theme Switching: Enable users to switch between dark and light themes for optimal reading experience.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) for a calm and reflective ambiance.
- Background color: Off-White (#F5F5F5) in light mode and Dark Gray (#333333) in dark mode.
- Accent color: Dusty Rose (#D8BFD8) for interactive elements.
- Headline font: 'Belleza', a humanist sans-serif that conveys personality and an artistic style.
- Body font: 'Alegreya', a humanist serif that complements 'Belleza', for an elegant reading experience.
- Use minimalist, line-based icons for categories and navigation.
- Clean, single-column layout for readability with a sidebar for navigation and search.
- Subtle fade-in animations on content loading and theme switching for a smooth user experience.
